## 📊 Super Store Sales Dashboard & Forecast (Power BI)
### 📌 Project Overview

This project presents an interactive Super Store Sales Dashboard and 15-Day Sales Forecast built using Power BI. The dashboard provides insights into sales performance, profit trends, customer segments, shipping modes, and regional distribution to help businesses understand their sales patterns and make better decisions.

The project also includes a sales forecasting model to predict sales trends for the next 15 days based on historical data.

### 🎯 Project Objective

The objective of this project is to:
- Analyze Super Store sales performance
- Identify top performing categories and regions
- Understand customer purchasing behavior
- Monitor monthly profit and sales trends
- Forecast future sales for the next 15 days

### 🛠 Tools & Technologies Used

- Power BI – Dashboard development
- Power Query – Data transformation
- DAX (Data Analysis Expressions) – Calculated measures
- Time Series Forecasting – Sales prediction
